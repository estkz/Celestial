This is a legacy project that I no longer plan to expand or maintain. I decided to revisit the source on March 13, 2026, purely to perform an emergency refactor because the original code was a horrific mess. It reached a point where I actually hated seeing it on my GitHub page; the technical debt was an eyesore and a poor representation of how I write code today.

While I’ve cleaned it up significantly, it still isn't perfect or fully up to my current standards. I’ve refactored it just enough to reach a state of tolerability that I can live with. To be clear, this doesn't mean the architecture follows SOLID principles or modern best practices—it’s just no longer an embarrassment to look at. It stays here now as a static archive and a reminder of where I started, finally meeting a baseline I'm comfortable showing publicly.

If you want to look at this shitty project, go ahead; you might still learn something from it regardless of the refactor being pretty rushed.

This project is x86, so build it using the correct architecture.
