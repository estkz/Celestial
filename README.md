Celestial is a tool designed to help game hackers calculate various aspects of a game that can be manipulated to gain an advantage over other players. One common example of this is the viewangle calculation, which involves determining the precise angle at which a player's character is looking in the game.

The calculator is capable of performing calculations using three different numeral systems: hexadecimal, decimal, and binary. This allows users to easily convert between different formats and perform complex calculations with ease.

The user interface of the calculator is created using the imgui library, which provides a lightweight, customizable, and responsive interface for the user. This allows users to quickly and easily input data, perform calculations, and view results without the need for complex programming or manual data entry. The interface is designed to be intuitive and user-friendly, with clear labels, easy-to-use controls, and helpful tooltips to guide the user through the process.
